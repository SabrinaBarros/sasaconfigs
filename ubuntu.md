## Package Manager 🗃

### NPM

```js
$ sudo apt install npm
```

### SNAP

```js
$ sudo apt-get install snapd snapd-xdg-open
```

<hr>

## GIT 🔗

```js
$ apt-get install git
```

## Node 🥝

- Install

```js
$ sudo apt install nodejs
```

- Remove

```js
$ sudo apt remove nodejs
```

## VS Code 👁️

- Install

```js
$ sudo snap install --classic vscode
```

- Att

```js
sudo snap refresh vscode
```

- Remove

```js
sudo snap remove vscode
```
